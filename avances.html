Avances del Proyecto

Aquí encontrarás los principales hitos alcanzados durante el desarrollo de Flex-IA.

Interfaz Gráfica en Java

Se diseñó la estructura visual de la aplicación con Swing, aplicando una interfaz moderna y adaptada a la guía de estilo del proyecto.

Ver más
Detallesclose

La interfaz incluye botones personalizados, menús interactivos y campos de texto con validación visual. Se respetó la paleta de colores establecida para mantener coherencia visual.

Reconocimiento de Manos

Implementación del módulo en Python con OpenCV y MediaPipe para detectar movimientos y posturas de la mano en tiempo real.

Ver más
Detallesclose

El sistema reconoce puntos clave (landmarks) y valida la correcta ejecución de los ejercicios. Se está trabajando en su integración con la interfaz Java mediante sockets.

Integración de Módulos

Se estableció comunicación entre los módulos de Python y Java para compartir información del reconocimiento de manos en tiempo real.

Ver más
Detallesclose

Mediante el uso de sockets TCP, se transmite la información procesada desde Python hacia la aplicación Java, permitiendo la visualización de resultados en la interfaz.
